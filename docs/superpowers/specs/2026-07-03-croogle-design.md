# Croogle — Design Spec

**Хакатон:** Worst Startup Ever (nFactorial) · 4-часовой спринт
**Дата:** 2026-07-03
**Tagline:** «Красота — для бедных. Настоящая роскошь — рукотворные каракули.»

## 1. Концепция

Croogle — поисковик, который **форвардит запрос в настоящий Google** и показывает
реальную выдачу, но **качество прорисовки интерфейса зависит от тарифа**. Настоящая
рабочая технология (Google Custom Search API) спрятана за абсурдной монетизацией:
пользователь платит не за функцию, а за **внешний вид** результатов.

Две сшитые «боли» (легенда для питча):
1. «Каждый день миллионы открывают Google и забывают, что хотели найти» → мы
   форвардим запрос напрямую, «убирая трение мысли».
2. «Красивый вылизанный поиск умеет каждый — он бесплатный и массовый. Настоящая
   роскошь — рукотворные каракули, нарисованные вручную. Мы первые, кто монетизировал
   эксклюзивность уродства.»

**Инверсия ценности (ядро абсурда):** чем ДОРОЖЕ тариф, тем ХУЖЕ выглядит интерфейс.
Бесплатный тариф — вылизанный «настоящий Google». Самый дорогой ($499) — кривые
каракули. Легенда: каракули «рисуются вручную artisan-художником под каждый запрос»,
это «limited edition», «premium exclusivity». Платишь больше — получаешь меньше пикселей.

**Удержание в теме (важно):** рабочий поиск сам по себе полезен (риск штрафа −5 за
«случайную пользу»). Спасает то, что **деньги берутся за эстетику, причём за её
УХУДШЕНИЕ** — это держит продукт глубоко в рамках темы. Питч делает упор именно на это.

## 2. Тарифная лестница (мемное ядро — цена растёт, качество падает)

Уровень оплаты = уровень «прорисованности» наоборот. Один и тот же поиск, разный рендер.

| Тариф | Цена | Визуал |
|---|---|---|
| **Шедевр** | $0 | Вылизанный «настоящий Google»: тени, тёмная тема, микроанимации, премиум-типографика |
| **Дизайн** | $9/мес | Чистый нормальный дизайн |
| **Набросок** | $49/мес | Плоский студенческий UX/UI: ровные блоки, один цвет, простые отступы |
| **Каракули** | $499/мес | **Каракули**: рукописный дрожащий шрифт (Shantell Sans), кривые рамки (rough.js), «набросок фломастером на салфетке» — «рисуется вручную под каждый запрос» |

Переключение тарифа перерисовывает интерфейс вживую — ключевой демо-момент: поднимаешь
цену на сцене, и Google «деградирует» в каракули под общий смех.

## 3. Архитектура (Next.js на Vercel)

```
Браузер ──▶ /api/search?q=...  ──▶ Google Custom Search JSON API
   ▲                                   (API-ключ спрятан на сервере)
   └── рендер результатов в теме текущего тарифа
```

- **Стек:** Next.js (App Router) + React, деплой на Vercel (обязательно — штраф −10 за localhost).
- **Скрытие ключа:** API-ключ Google и Search Engine ID живут только в серверных env-переменных;
  фронт их не видит. Serverless API-роут решает и CORS, и безопасность ключа.

## 4. Компоненты

1. **Landing/Hero** — заголовок, CTA «Найти красиво», фейк-метрики, pricing-таблица,
   отзывы, блок «о нас в СМИ», FAQ.
2. **SearchBar** — поле ввода, отправляет запрос на `/api/search`.
3. **ResultsList** — рендерит нормализованную выдачу в теме текущего тарифа.
4. **ThemeProvider** — хранит `tier`, применяет класс темы на корневой элемент.
   Те же компоненты результатов, 4 набора стилей.
5. **FakePaywall** — модалка апгрейда с тарифами + псевдо-Stripe чекаут, который
   **ничего не списывает, а просто переключает тему**.

## 5. Data Flow

1. Пользователь вводит запрос → `SearchBar` шлёт `GET /api/search?q=...`.
2. `/api/search` зовёт Google Custom Search JSON API, нормализует ответ в
   `{ title, link, snippet }[]`.
3. **Fallback:** при ошибке или превышении лимита (100 запросов/день) роут отдаёт
   **захардкоженный кэш** заранее заготовленных ответов → демо не падает (бонус +3).
4. `ResultsList` рендерит массив в стилях текущего `tier`.

## 6. Монетизация (буллшит-блок)

- **Ugliness-as-a-Service (UaaS).** Шедевр $0 · Дизайн $9 · Набросок $49 · Каракули $499/мес
  («самый дорогой тариф выглядит хуже всех — в этом и роскошь»).
- Фейк-метрики: «12.4M запросов отрендерено вручную», «средний Каракули-пользователь
  платит в 500 раз больше за в 10 раз меньше пикселей», «retention каракулей 97%».
- «Взрослые» детали (добивают в блоке проработки): privacy policy («мы не храним ваши
  запросы, только ваш вкус»), status page uptime 99.99%, вакансия *Senior Scribble Artist*.

## 7. Scope / YAGNI (за 4 часа)

**В скоупе:** один поиск-эндпоинт + кэш-fallback, 4 темы, переключатель тарифа,
лендинг с копирайтом, фейк-чекаут, деплой на Vercel.

**Вне скоупа:** реальные платежи, аккаунты/авторизация, база данных, история поиска.
Тариф — просто состояние в браузере (React state / localStorage).

## 8. Критерии успеха (маппинг на 100 баллов)

- **Техника (30):** живой Google Custom Search API + fallback, стабильное демо.
- **Лендинг/визуал (20):** 4 темы, pricing, FAQ, отзывы, «в СМИ», рабочий переключатель.
- **Креатив/абсурд (20):** оригинальный угол «платишь за красоту, а не функцию», мемность.
- **Питчинг (20):** каменное лицо, «Красота — для бедных, роскошь — каракули», ответы
  на «как масштабируетесь / в чём монетизация».
- **Буллшит (10):** UaaS, TAM, фейк-метрики, Каракули-тариф $499, вакансия, status page.
